示例计算

模式1示例：借10000，利10%，抵押1000，每期还10%，4期

利息 = 10000 × 10% = 1000

到手金额 = 10000 - 1000 - 1000 = 8000

实际出资金额 = 8000（我们的本钱）

每期还本金 = 10000 × 10% = 1000

每期还款 = 1000 + 1000 = 2000（第一期包含全额利息）

期数 = 4期（可自定义）

总还款 = (2000 × 4) - 1000 = 7000

公司利润 = 7000 - 8000 = -1000（基于实际出资金额）

模式2示例：借10000，利10%，无抵押，4期

利息 = 10000 × 10% = 1000

到手金额 = 10000 - 1000 = 9000

实际出资金额 = 9000（我们的本钱）

每期还款 = (10000 + 1000) ÷ 4 = 2750

期数 = 4期（可自定义）

总还款 = 2750 × 4 = 11000

公司利润 = 11000 - 9000 = 2000（基于实际出资金额）
